Breathe: Meditation App

Breathe is a meditation app that uses your breathing patterns to suggest a custom meditation for relaxation. Built with Angular, this app features user profiles secured with Node Authorization, interactivity, and storage of user data with MongoDB. It's core components use vanilla JS.

  • HTML5/CSS3
  • AngularJS
  • Node/Express
  • Node Authorization
  • MongoDB/Mongoose

Quote App

This app is a random quote generator built with Angular and Vanilla JS. You can get a random quote adn save them to a list. If random quotes aren't your thing, you can search all quotes by author or genre. You can even add your own quotes to the repository.

  • HTML5/CSS3
  • AngularJS
  • Vanilla JS

Sunset

This app uses geolocation to tell you that day's sunset time. It queries an external API using AJAX and Vanilla JS to reveal the time. It features a fun CSS animation. It is also the prototype for a larger hardware project that aims to synchronize bed lamps with the rising and setting of the sun.

  • HTML5/CSS3
  • Vanilla JS
  • AJAX
  • External APIs
  • Geolocation
